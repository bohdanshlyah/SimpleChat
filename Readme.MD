### Local running BE
#### Dependencies
* python 3.9
* redis 
* postgresql

Install python requirements
```
cd ./chat-be/
pip install -r requirements.txt
```

Run migrations:
```
python manage.py migrate
```

Run application:
```
python manage.py runserver  0.0.0.0:8000
```

### Local running frontend (FE)

#### Requirements
* node 16

Install node modules
```
npm install
```

Run chat-ui dev application localy:
```
npm run dev
```
